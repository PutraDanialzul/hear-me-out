#Hear Me Out 🌱

Hear Me Out is a student-centered platform designed to support emotional well-being through anonymous confessions, private reflection, and safe self-expression.

The platform provides a space where users can anonymously share confessions with the community while also maintaining a private journal of thoughts, emotions, and personal experiences.

---

## ✨ Features

### Anonymous Confessions

- Share thoughts anonymously
- Browse confessions posted by others
- Search confessions
- Sort by newest or oldest
- Pagination support
- Report inappropriate content

### Mind Space

A private personal journal designed for reflection.

- Create private entries
- Search personal entries
- Sort entries
- Store thoughts securely
- Accessible only to the owner

### Moderation System

To help maintain a safe environment:

- Report submission system
- Moderation queue
- Content verification
- Report management
- Moderator-only tools

### Authentication

- Supabase Authentication
- Secure user sessions
- Account-based ownership
- Role-based moderation access

### User Experience

- Responsive design
- Light mode
- Dark mode
- Mobile-friendly layouts
- Open Graph previews
- SEO metadata support
